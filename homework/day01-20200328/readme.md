2. 猜数字游戏
    生成随机整数[0, 100)
    提示用户再控制台输入猜测的数字
    比较，当用户输入较大，提示太大了
        当用户输入太小，提示太小了
        当用户输入正确，提示经过N次对了，太聪明了
    用户最多猜5次，如果5次内都没有猜正确，提示太笨了，游戏结束

3. 扩展: 当成功或失败后，提示用户是否继续，输入：yes, y, Y则继续，重新生成随机数，让用户猜测
